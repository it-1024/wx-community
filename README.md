# wx-community
> 微信社区/信息发布类小程序 [点击进入官网](http://m.it-1024.net)





### 基本功能（V1版本）
- [x]  相关信息搜索
- [x]  获取定位信息
- [x]  发布文字与图片信息
- [x]  查看详细信息
- [x]  分享、评论、点赞、拨打联系电话等、
